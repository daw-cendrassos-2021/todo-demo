# Exemple de cookies i de docblock

Todo - Demo
===========

Petita demo d'un gestor de tasques, per demostrar com funcionen les cookies. 

M07 - Programació d'aplicacions web - UF1.
M08 - Programació d'aplicacions web - UF4.


- La v0.1 és amb PHP "planer".
- La v1.0 utilitza un model simple per gestionar les tasques.
- La v1.1 utilitza un model SQLite.
- La v1.2 utilitza un model MySQLi.
- La v1.3 utilitza un model PDO.
- La v2.0 Implementa un MVC molt simple amb controladors basats en funcions i els model que ja haviem creat.


A la primera versió per desar la informació utilitza una cookie.
